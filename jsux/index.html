Shortcuts

  • Optimized UX for the very common cases
  • Steep learning curve; even slightly more complex cases need to go the complex route

Optional arguments


				// Common case: standard, minified JSON
				JSON.stringify(data);
			

				// Complex case: custom serialization, formatting
				JSON.stringify(data, (key, value) => {
					if (value instanceof HTMLElement) {
						return value.outerHTML;
					}

					return value;
				}, "\t");

Function arguments


				// Common case: replace with string
				text.replaceAll(/\s+/g, " ");
			

				// Complex case: replace with function
				text.replaceAll(/-[a-z]/g, match => match.toUpperCase());
